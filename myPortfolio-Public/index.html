Fifa Algorithm

×
About

Algorithm that receives a data input and does the processing returning the output.

The input consists of several stickers (id, quantity) that should complete a sticker book. The first series corresponds to the user's stickers, the rest being friends' stickers.

The algorithm will check which friend has more stickers to trade ("greedy") and how many possible trades there are, making the trade. The output consists of the number of trades made and with which friend the trade was carried out.
